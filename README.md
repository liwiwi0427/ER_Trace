# ⚠️聲明⚠️
## 在使用前請 「*務必詳閱*」 以下聲明
 1. 僅供教學用途
本網站（ER Trace，或「急診心跡」）所提供之心電圖波形、生理數據、藥物反應及相關說明，僅供醫學教育、模擬演練與學術交流參考使用。<br>
**本網站之設計目的在於輔助學習，而非作為最終診斷依據。**    
 2. 非醫療診斷工具
本網站所載之任何資訊絕非醫療建議，不可替代專業醫療人員之臨床決策、判斷與處置，使用者不應依賴本系統之內容對實際病患進行診斷、治療、給藥或執行任何醫療處置。<br>    
**若有任何醫療疑慮，請洽醫師或是相關專家！**
 3. 臨床處置原則
實際臨床情境極為複雜且具個體差異，在面對真實病患時，請務必依據所在醫療機構之作業流程、現行醫療法規，並諮詢專科醫師或醫療團隊之專業意見進行處置。
4. 資料準確性與更新
本網站盡力確保所引用之急救準則（如 AHA ACLS Guidelines）與病理資訊之正確性；然而，醫學知識與治療指引不斷演進，開發者不保證本系統內所有資訊均為最新或完全無誤。<br>
**若有與現行臨床指引不符之處，應以權威醫療機構發布之最新指引為準。**
5. 責任限制
開發者對於因使用或無法使用本系統所導致之任何直接、間接、意外或衍生之損害（包括但不限於學習誤解、考試失利或臨床處置不當），概不負任何法律責任。<br>
**使用者應自行承擔使用本系統之所有風險。**
6. 緊急狀況
若您或他人處於緊急醫療狀況，請立即停止使用本軟體，並撥打緊急救護電話（如 119）或前往最近的醫院急診室求助。
